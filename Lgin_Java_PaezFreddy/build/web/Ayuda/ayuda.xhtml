El sistema contiene los siguientes apartados:
  • Seguridad
  • Personal
  • Proyectos
  • Finanzas
En Seguridad es posible:
  • Realizar el CRUD de los Usuarios
  • Realizar el CRUD de los Perfiles de Usuarios
  • Realizar el CRUD de las Opciones por Menú
  • Asignar a los usuarios un perfil de Usuario
  • Asignar las opciones por perfil
En Personal es posible:
  • Realizar el CRUD de los Empleados
En Reportes ses posible:
  • Visualizar los Usuarios por Géneros
  • Visualizar los Usuarios por Departamentos
  • Visualizar los Usuarios por localización
  • Visualizar los Usuarios por fechas
